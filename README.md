# Navegação Completa com Formulário usando AsyncStorage

Este projeto é um exemplo de como implementar uma navegação completa em um aplicativo React Native, utilizando **AsyncStorage** para persistência de dados e um formulário para entrada de informações.

## Funcionalidades

- Navegação entre telas com **React Navigation**.
- Persistência de dados local com **AsyncStorage**.
- Formulário para entrada e armazenamento de dados do usuário.

## Tecnologias Utilizadas

- **React Native**
- **React Navigation**
- **AsyncStorage**

## Pré-requisitos

Antes de começar, certifique-se de ter as seguintes ferramentas instaladas:

- Node.js
- React Native CLI ou Expo CLI
- Editor de código (recomendado: Visual Studio Code)
